# vuejs-stripe-menu
Vuejs version of stripe.com top menu.

![Demo vuejs stripe menu](/vuejs-stripe-menu-demo.gif "Demo")


## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
